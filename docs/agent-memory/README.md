# Agent memory (scrubbed)

Point-in-time engineering notes accumulated by the coding agent across this project
(catalog + noice on OHOS via the appspawn-x adapter). `MEMORY.md` is the index;
each topic file is one finding/recipe. They are *observations at the time of writing* —
verify file/line/flag claims against current sources before relying on them.

**Privacy:** these are SCRUBBED for public release. Removed/redacted: real names,
emails, the remote build-server hostnames/IPs and credentials, and absolute user
paths (`/home/<user>` → `$HOME`, Windows user paths genericized). Four files that
were purely remote-build-infra/credentials were omitted entirely (their `[[links]]`
in MEMORY.md are intentionally dangling). Tokens like `[REDACTED-CRED]`,
`[REDACTED-HOST]`, `[REDACTED-EMAIL]`, `[build-host]`, `[operator]`, `[user]`, `[id]`
mark where private values were removed.
